# Theme 主题设置

## 参数设置
- colorPrimary
主要品牌颜色，一般用于ActionBar背景

- colorPrimaryVariant
主要品牌颜色的可选颜色

- colorPrimaryDark
默认用于顶部状态栏和底部导航栏

- colorOnPrimary
在主要品牌颜色上面的东西设置的颜色

- colorAccent
默认设置给colorControlActivated，一般是主要品牌颜色的明亮版本补充。`光标颜色`

## 主题选取
`Theme.MaterialComponents.DayNight.NoActionBar`和`Theme.MaterialComponents.DayNight.NoActionBar.Bridge`

后者继承了`AppCompat`
